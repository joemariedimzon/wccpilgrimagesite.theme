wccpilgrimagesite.theme Installation
------------------------------------

To install wccpilgrimagesite.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wccpilgrimagesite.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wccpilgrimagesite.theme

* Re-run buildout, e.g. with:

    $ ./bin/buildout
